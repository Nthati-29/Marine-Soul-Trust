# Marine-Soul-Trust
 Project Overview
This repository contains the web development project for Marine Soul Trust , a non-profit organization focused on marine conservation, coral reef rehabilitation, and coastal education. 
Mission Statement: We're committed to safeguarding our oceans, ensuring that marine life flourishes in a clean, plastic-free environment.

This project serves as a centralized digital hub for the organization, bridging the gap between local community conservation efforts in South Africa and global marine advocacy. This project was developed as a submission for the WEDE5020 Portfolio of Evidence (PoE) module.

Design and Planning
The development process followed an iterative design approach:
- Wireframing & Sitemap: The website’s structural architecture (Sitemap) and user interface blueprint (Wireframe) were designed using Figma. This allowed for a mobile-responsive "Global-Local" design strategy before the coding phase began.
  
  Project Structure
The project follows a multi-page structure to ensure clear navigation, improved SEO, and a better user experience.

├── index.html            Homepage (Hero and Mission)
├── about.html            About Us (History, Vision, Team)
├── services.html         Projects (Educational Resources, Sponsorships, Merchandise)
├── enquiry.html          Volunteer/Partnership Form
├── contact.html          Contact Information (Locations)
├── css/
│   └── style.css         Master Stylesheet (Responsive Design)
├── images/               Asset storage
└── README.md             Project documentation
 Technologies Used
Languages: HTML5 (Semantic Markup), CSS3 (Flexbox/Grid).
Design Tool:Figma (Wireframing and Sitemap architecture).
Development Environment: Visual Studio Code (VS Code).
 Setup and Installation
 1. Clone or download this repository to your local machine.
 2. Open the project folder in Visual Studio Code.
 3. Use the Live Server extension to launch the website in your browser.
 4. Ensure all pages (index.html, about.html, services.html, enquiry.html, contact.html) and the css/ directory are in the same parent folder to maintain link integrity.
Key Features
Responsive Navigation: A unified navigation bar across all five pages using semantic <nav> elements.
 Service-Based Portal: A dedicated page (services.html) offering educational toolkits, sponsorship opportunities for donors, and sustainable merchandise.
 Multi-Location Contact: A contact page featuring two distinct operational locations (Cape Town and Durban).
 Mobile-First Design: CSS media queries ensure the layout adapts seamlessly to mobile, tablet, and desktop screens.
